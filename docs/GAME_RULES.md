# GAME RULES ? Yacht (요트 다이스 게임)

## 개요
- 게임 이름: Yacht (요트 스타일 주사위 게임)
- 참가자: 최대 4명
- 목표: 모든 플레이어가 자신의 점수표를 모두 채운 뒤, 총점이 가장 높은 플레이어가 승리합니다.

## 기본 진행
1. **플레이어 및 순서**
   - 최대 4명의 플레이어가 참가할 수 있습니다.
   - 플레이어들은 시계 방향으로 한 명씩 차례를 갖습니다.

2. **한 플레이어의 턴 진행**
   - 자신의 차례가 되면 해당 플레이어는 주사위 5개를 굴립니다(1회째 굴림).
   - 플레이어는 최대 총 3번까지 주사위를 굴릴 수 있습니다(초기 굴림 포함). 즉, 추가로 최대 2회까지 더 굴릴 수 있습니다.
   - 각 추가 굴림 전에 플레이어는 원하는 주사위를 고정(keep)하거나 해제할 수 있습니다. 고정한 주사위는 다음 굴림에서 굴려지지 않습니다.
   - 플레이어는 3번의 기회를 모두 사용하지 않고 중간에 굴림을 멈출 수 있습니다(원할 때 점수를 기록할 수 있음).
   - 주사위 굴림을 멈추면(혹은 3번 굴린 후) 플레이어는 자신의 점수표에서 아직 비어 있는 한 칸을 선택하여 해당 카테고리에 결과를 기록합니다. 조건을 만족하지 않으면 0점을 기록할 수 있습니다.

3. **라운드와 게임 종료**
   - 모든 플레이어가 한 번씩 차례를 마치면 1라운드가 종료됩니다.
   - 각 플레이어가 자신의 점수표의 모든 항목(카테고리)을 채우면 게임이 종료됩니다. 즉, 점수표에 남은 빈칸이 없을 때까지 진행합니다.

## 점수판(카테고리) 및 채점 규칙
점수표는 크게 표준(Upper) 항목과 스페셜(Special) 항목으로 구성됩니다. 각 항목의 채점 규칙은 아래와 같습니다.

1) **스탠다드(Upper) ? One through Six**
   - **원탑 (Ones)**: 주사위 중 '1'인 것들의 합계를 기록합니다. (예: 주사위가 [1,1,3,5,6]이면 1+1 = 2점)
   - **투탑 (Twos)**: '2'의 합계를 기록합니다. (예: [2,2,2,4,6] → 2+2+2 = 6점)
   - **쓰리탑 (Threes)**: '3'의 합계
   - **포탑 (Fours)**: '4'의 합계
   - **파이브탑 (Fives)**: '5'의 합계
   - **식스탑 (Sixes)**: '6'의 합계

2) **스페셜(Special)**
   - **트리플 (Three of a Kind)**: 동일한 숫자가 3개 이상 있을 경우, 그 같은 숫자 3개의 합을 점수로 기록합니다. (예: [4,4,4,2,6] → 4+4+4 = 12점)
   - **포카드 (Four of a Kind)**: 동일한 숫자가 4개 이상 있을 경우, 그 같은 숫자 4개의 합을 점수로 기록합니다. (예: [5,5,5,5,2] → 5+5+5+5 = 20점)
   - **풀하우스 (Full House)**: 같은 숫자 3개와 다른 같은 숫자 2개(3+2 구조)를 만족하면, 해당 다섯 주사위의 총합(모든 주사위 합)을 기록합니다. (예: [3,3,3,6,6] → 3+3+3+6+6 = 21점)
   - **스몰 스트레이트 (Small Straight)**: 연속된 네 개 숫자(예: 1-2-3-4, 2-3-4-5, 3-4-5-6)가 주사위에 포함되어 있으면, 그 연속을 구성하는 네 주사위의 합을 기록합니다. (예: 주사위에 2,3,4,5가 있으면 해당 주사위들의 합)
   - **라지 스트레이트 (Large Straight)**: 연속된 다섯 개 숫자(1-2-3-4-5 또는 2-3-4-5-6)가 나온 경우, 다섯 주사위의 합을 기록합니다.
   - **초이스 (Choice)**: 어떤 조합이든 상관없이, 다섯 주사위의 합을 기록합니다.
   - **요트 (Yacht)**: 다섯 주사위가 모두 동일한 숫자인 경우 고정 점수 50점을 부여합니다.

※ 각 스페셜 항목을 선택할 때 조건을 만족하지 않으면 해당 칸에 0점을 기록해야 합니다.

## 점수 집계 및 승리 조건
- 모든 플레이어가 자신의 점수표의 모든 항목을 채우면 각 플레이어의 총점을 계산합니다.
- 총점이 가장 높은 플레이어가 우승합니다. 동점일 경우 동점으로 처리(추가 규칙을 정하고 싶으면 별도 합의).

## 예시 흐름 요약
1. 플레이어 A가 차례 → 5개 주사위 굴림(1회)
2. 원하는 주사위를 고정하고(또는 모두 해제) 2회 더 굴림(선택적으로 중간에 멈출 수 있음)
3. 굴림을 멈추면 점수표의 한 항목을 선택해 점수를 기록(조건 불충족시 0점)
4. 모든 플레이어가 차례를 마치면 라운드 종료
5. 각 플레이어가 모든 항목을 채우면 게임 종료 → 총점 비교
