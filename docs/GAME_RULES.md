# GAME RULES - Yacht Dice (야찌 다이스 게임)

## 개요
- 게임 이름: Yacht (야찌 스타일 주사위 게임)
- 플레이어: 1~4명
- 목표: 모든 플레이어가 자신의 점수표를 모두 채운 후, 점수의 합이 가장 높은 플레이어가 승리합니다.

## 기본 규칙

### 1. 플레이어 및 순서
- 최대 4명의 플레이어가 참여할 수 있습니다.
- 플레이어들은 시계 방향으로 돌아가며 턴을 진행합니다.

### 2. 각 플레이어의 턴 진행
- 자신의 턴이 되면 해당 플레이어는 주사위 5개를 굴립니다(1회차 굴림).
- 플레이어는 최대 총 3번까지 주사위를 굴릴 수 있습니다(초기 굴림 포함). 즉, 추가로 최대 2회까지 더 굴릴 수 있습니다.
- 각 추가 굴림 전에 플레이어는 원하는 주사위를 유지(keep)하거나 풀 수 있습니다. 유지된 주사위는 다음 굴림에서 굴려지지 않습니다.
- 플레이어는 3번의 기회를 모두 사용하지 않고 중간에 점수를 선택할 수 있습니다(언제든 선택이 가능).
- 주사위 굴림을 마치면(혹은 3번 모두 쓴 후) 플레이어는 자신의 점수표에서 아직 비어 있는 한 칸을 선택하여 해당 카테고리에 점수를 기록합니다. 조건을 만족하지 못하면 0점으로 기록할 수 있습니다.

### 3. 라운드 및 게임 종료
- 모든 플레이어가 한 번씩 턴을 마치면 1라운드가 종료됩니다.
- 각 플레이어가 자신의 점수표의 모든 항목(카테고리)을 채우면 게임이 종료됩니다. 즉, 점수표에 빈칸이 없으면 게임이 끝납니다.
- 총 13라운드 진행 (13개 카테고리)

## 점수판(카테고리) 및 채점 규칙

점수표는 크게 스탠다드(Upper) 항목과 스페셜(Special) 항목으로 나뉩니다.

### 1) 스탠다드(Upper) - Ones ~ Sixes

| 카테고리 | 설명 | 점수 계산 |
|----------|------|-----------|
| Ones (에이스) | 주사위 중 '1'의 합계 | 1 x 개수 |
| Twos (듀스) | 주사위 중 '2'의 합계 | 2 x 개수 |
| Threes (트리플) | 주사위 중 '3'의 합계 | 3 x 개수 |
| Fours (쿼드) | 주사위 중 '4'의 합계 | 4 x 개수 |
| Fives (파이브) | 주사위 중 '5'의 합계 | 5 x 개수 |
| Sixes (식스) | 주사위 중 '6'의 합계 | 6 x 개수 |

**스탠다드 보너스**: 스탠다드 섹션(1~6) 합계가 **63점 이상**이면 **+35점 보너스**를 획득합니다.

### 2) 스페셜(Special)

| 카테고리 | 조건 | 점수 |
|----------|------|------|
| Three of a Kind | 같은 숫자 3개 이상 | 해당 숫자 x 3 |
| Four of a Kind | 같은 숫자 4개 이상 | 해당 숫자 x 4 |
| Full House | 3개 + 2개 조합 | 모든 주사위 합 |
| Small Straight | 연속 4개 (1-2-3-4, 2-3-4-5, 3-4-5-6) | **15점 (고정)** |
| Large Straight | 연속 5개 (1-2-3-4-5 또는 2-3-4-5-6) | **30점 (고정)** |
| Choice | 제한 없음 | 모든 주사위 합 |
| Yacht | 5개 모두 동일 | **50점 (고정)** |

※ 각 조건을 만족하지 못해도 해당 칸에 0점으로 기록할 수 있습니다.

## 점수 계산 예시

### 스탠다드 섹션
- 주사위 [1,1,3,5,6] → Ones 선택 시: 1+1 = **2점**
- 주사위 [2,2,2,4,6] → Twos 선택 시: 2+2+2 = **6점**

### 스페셜 섹션
- 주사위 [4,4,4,2,6] → Three of a Kind: 4x3 = **12점**
- 주사위 [5,5,5,5,2] → Four of a Kind: 5x4 = **20점**
- 주사위 [3,3,3,6,6] → Full House: 3+3+3+6+6 = **21점**
- 주사위 [1,2,3,4,6] → Small Straight: **15점**
- 주사위 [2,3,4,5,6] → Large Straight: **30점**
- 주사위 [6,6,6,6,6] → Yacht: **50점**

## 게임 종료 및 승리 조건
- 모든 플레이어가 자신의 점수표의 모든 항목을 채우면 각 플레이어의 점수를 합산합니다.
- 스탠다드 섹션 합계가 63점 이상이면 35점 보너스가 추가됩니다.
- 점수의 합이 가장 높은 플레이어가 승리합니다.
- 동점일 경우 무승부 처리됩니다.

## 게임 흐름 예시
1. 플레이어 A의 턴 → 5개 주사위 굴림 (1회)
2. 원하는 주사위를 유지하고 (또는 모두 굴려) 2회차 굴림 (선택적으로 추가 굴림 가능)
3. 점수표의 빈 항목을 선택하여 점수 기록 (조건 미충족 시 0점)
4. 모든 플레이어가 턴을 마치면 다음 라운드
5. 각 플레이어가 모든 항목을 채우면 게임 종료 후 최종 점수 계산

## UI 기능

### 주사위 표시
- 게임 시작 시 주사위는 [6,6,6,6,6]으로 표시됨
- 턴이 넘어갈 때 이전 주사위 값이 유지되어 표시됨 (굴리기 전까지)
- 다른 플레이어가 주사위를 굴릴 때도 흔들리는 애니메이션과 사운드 효과 재생

### 채팅 시스템
- 6개 카테고리: 칭찬, 도발, 응원, 반응, 인사, 감정
- 각 카테고리당 6개의 빠른 메시지 선택 가능
- 육각형 배치 UI

### 사운드
- BGM: 홈페이지부터 자동 재생
- 효과음: 주사위 굴리기, 버튼 클릭, 야찌 달성

- 난이도별 AI 플레이어 추가 가능
- 상단 보너스(63점) 달성을 위한 전략적 플레이
- 기대값 기반 의사결정 알고리즘
